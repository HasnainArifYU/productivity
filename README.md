# Productivity Application

A modern Spring Boot application for managing notes and to-do lists with remote access.

## Features (Planned)

- User authentication and authorization
- Notes management with tags/categories
- To-do list management with priorities and due dates
- RESTful API design
- Secure remote access

## Technology Stack

- Java 17+
- Spring Boot 3.x
- Spring Data JPA
- Spring Security with JWT
- PostgreSQL/H2 Database
- Docker containerization
- CI/CD pipeline
